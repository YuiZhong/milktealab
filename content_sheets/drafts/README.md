# Content Sheet Drafts｜制作期草案工作区

本目录用于放置制作期草案表格，供用户、ChatGPT 和 Codex 后续审阅。

`ingredient_profile_value_draft.v0.0.8.11.csv` / `.json` 覆盖全部 37 个原料，导出现有 taste / texture / flavor profile 快照，并预留 source research / review 字段。

这些文件不是 runtime data，不是 generated data，不是 source-of-truth，不能直接导入游戏。

本轮 `proposed*` 字段保持空对象；后续必须经过联网资料 / 食品资料 / 人类 review 后，才能填写 proposed values。

CSV 使用 UTF-8 with BOM，方便 Excel / Numbers / Google Sheets 打开中文。
