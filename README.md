# 奶茶实验室

一个原创饮品研发经营游戏原型。当前阶段是配方实验室测试版。

## 当前版本

v0.0.4.0

## 运行方式

可以直接打开 `index.html`，或在项目目录运行：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173/
```

## 当前功能

- 自由添加原料
- 调整配方比例
- 自动配平
- 试喝报告
- 饮品类型判断
- 属性条
- 保存配方
- 查看已保存配方
- 本地浏览器存档

## 部署方式

这是纯静态网页项目，可部署到 GitHub Pages / Netlify / Vercel。

入口文件为 `index.html`。
