# Test Cases

## 1. 当前 v0.0.4.x 回归测试原则

v0.0.4.x 只做工程整理，不应改变任何玩法表现。测试重点是确认没有回退。

## 2. 必测项目

### 页面打开

- 页面能正常打开
- 顶部版本号显示正常
- 控制台无 error

### 原料按钮

- 点击未加入原料，应加入当前杯子
- 再次点击已加入原料，应从当前杯子移除
- 不应出现每点一次 +10% 的累加逻辑
- 已选状态高亮 / ✓ 正常
- 删除、清空、随机乱摇、载入保存配方后，左侧按钮状态同步正常

### 比例条

- 手动改一个原料时，其他原料不动
- 总量可以低于 100%
- 总量不能超过 100%
- 总量为 100% 时，新加入原料应以 0% 或安全方式加入，不应突破 100%

### 核心功能

- 试喝一下可正常运行
- 随机乱摇可正常运行
- 清空杯子可正常运行
- 保存配方可正常运行
- 查看已保存配方可正常运行

## 3. 当前基础测试配方

### 清爽水果茶

绿茶 40 + 气泡水 30 + 柠檬 20 + 蜂蜜 10

预期：清爽水果茶，高分。

### 经典奶茶

红茶 45 + 牛奶 40 + 珍珠 15

预期：经典奶茶，中高分，反馈稳定，不要说清爽。

### 高级厚乳款

乌龙茶 40 + 厚乳 35 + 奶盖 20 + 海盐 5

预期：高级厚乳款，中高分，反馈体现高级感。

### 极端柠檬事故

柠檬 85 + 绿茶 5 + 气泡水 5 + 蜂蜜 5

预期：酸度事故，低分，反馈直接毒舌。

### 极端榴莲事故

榴莲 80 + 牛奶 10 + 红茶 10

预期：香气冲击 + 厚重 + 黏腻 + 一定吸管阻力。

### 奶脂过载

厚乳 70 + 淡奶油 20 + 植脂奶 10

预期：奶脂过载 / 高油腻事故。

### 吸管阻力事故

芋泥 45 + 奥利奥碎 32 + 珍珠 16 + 蜂蜜 7

预期：吸管阻力事故，低分，反馈说吸不上来 / 人生阻力 / 需要勺子。

### 气泡奶油冲突

气泡水 50 + 淡奶油 40 + 蜂蜜 10

预期：气泡和奶油冲突，不要说水泥。

### 工业奶茶

植脂奶 70 + 红茶 20 + 黑糖 10

预期：工业奶茶 / 廉价感 / 健康疑虑方向。

## 4. 未来味觉系统测试方向

v0.0.5.x 之后逐步加入：

- 基础经典饮品测试
- 极端事故测试
- 边界比例测试
- 多水果茶 / 花果茶泛化测试
- 温度 / 冰量 / 糖度测试
- 客群偏好测试
- 随机压力测试
- 金标样本回归测试

## 5. 已知限制

- 当前保存配方功能使用浏览器本地存储。
- 旧保存配方可能因打开地址、端口、域名、`file://` 与 `localhost` 切换、浏览器清理数据等原因不可见。
- 这是原型期已知限制，不在 v0.0.4.x 修复。
- 后续可考虑导出 / 导入 JSON、存档版本号和正式本地存档方案。
